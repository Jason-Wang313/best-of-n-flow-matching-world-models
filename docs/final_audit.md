## Novelty Audit

The repo does not claim a broad new method. The strongest honest claim is that proxy-only Best-of-N can select off-manifold trajectories from a rectified-flow world model and that feature-space uncertainty is a useful diagnostic/repair in the synthetic setting.

## Known Limitations

- Synthetic task only.
- Proxy model is intentionally misspecified.
- No diffusion baseline yet.
- No standard control benchmark yet.
- No learned uncertainty estimator yet.
